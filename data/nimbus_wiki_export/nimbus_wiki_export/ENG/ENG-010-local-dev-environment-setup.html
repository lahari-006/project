Local Dev Environment Setup


git clone git@github.com:nimbuscloud/platform.git
cd platform
make bootstrap   # installs deps, sets up pre-commit hooks
docker compose up -d   # starts local postgres, redis, kafka
make run SERVICE=billing-service

Local services are available at http://localhost:8080 by default. See individual service runbooks for service-specific env vars.
