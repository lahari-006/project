API Authentication Guide (Legacy API Keys)

Legacy method. API keys were fully deprecated and no longer function as of the OAuth2 migration.

Overview

API calls were authenticated using a static API key passed in the header:


curl https://api.nimbuscloud.io/v1/billing \
  -H "X-API-Key: nk_live_abc123..."

Keys did not expire and were rotated manually by request to the platform team.
