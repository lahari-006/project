API Authentication Guide

Current authentication standard for all Nimbus APIs, effective since the OAuth2 migration.

Overview

All external API calls must use OAuth 2.0 with the client_credentials grant. Static API keys were deprecated and disabled for all external partners.


curl -X POST https://auth.nimbuscloud.io/oauth/token \
  -d "grant_type=client_credentials" \
  -d "client_id=$CLIENT_ID" \
  -d "client_secret=$CLIENT_SECRET"

Access tokens expire after 1 hour and must be refreshed by re-requesting a new token.
