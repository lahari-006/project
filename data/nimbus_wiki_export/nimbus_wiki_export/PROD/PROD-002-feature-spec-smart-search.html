Feature Spec: Smart Search

Problem

Current search relies on exact keyword matching, leading to poor recall for typos and synonyms.

Solution

Introduce hybrid semantic + keyword search backed by a vector index, with typo tolerance and synonym expansion.

Success Metrics

  • Search-to-click rate +15%
  • Zero-result-rate reduced below 3%
