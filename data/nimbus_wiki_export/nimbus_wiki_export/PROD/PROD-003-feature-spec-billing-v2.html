Feature Spec: Billing v2

Overview

Billing v2 introduces usage-based billing alongside the existing flat-rate plans, and consolidates invoicing into a single monthly cycle per customer.

Key Changes

  • Support for metered usage line items
  • Proration handling for mid-cycle plan changes
  • Self-serve plan downgrade (previously required support ticket)
